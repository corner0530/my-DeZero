# My DeZero

[ゼロから作る Deep Learning ❸](https://www.oreilly.co.jp/books/9784873119069/)を読みながら，[DeZero](https://github.com/oreilly-japan/deep-learning-from-scratch-3)を各ステップごとに実装していく．

## 環境

- Python 3.10

## 注意事項

DeZero をパッケージとしてインストールするため，最初に以下を実行する．

- conda の場合
  ```bash
  conda develop .
  ```
- pip の場合
  ```bash
  pip install -e .
  ```

アンインストールする場合は

- conda の場合
  ```bash
  conda develop -u .
  ```
- pip の場合
  ```bash
  pip uninstall .
  ```
